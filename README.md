# 📊 PowerBI-Server-Log-Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard developed to analyze server log data. It helps monitor system performance, identify errors, and gain actionable insights using Power BI, Excel, and DAX.

---

## 🚀 Tools & Technologies
- Power BI
- Microsoft Excel
- DAX
- Data Cleaning
- Data Visualization

---

## 📈 Dashboard Features
- Executive Dashboard
- School Analytics
- Student Analytics
- Admission Dashboard
- Fee Analytics
- Birthday Dashboard

---

## 📊 Key KPIs
- Total Students
- Total Fee Collection
- Pending Fees
- New Admissions
- Revenue Analysis
- Student Distribution

---

## 🎯 Key Insights
- Analyze student admissions and fee collection.
- Track revenue trends across schools.
- Monitor student distribution and demographics.
- Interactive slicers and drill-down reports.
